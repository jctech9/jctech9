## Projetos em destaque

### Projeto Exemplo JSON para XLSX

Projeto de exemplo em Spring Boot que consulta uma API HTTP e transforma dados JSON em planilhas XLSX com Apache POI. A integração valida o destino da API e aplica proteções contra SSRF, incluindo allowlist, bloqueio de endereços não públicos e redirecionamentos desativados.

**Tecnologias:** Java 21, Spring Boot, Apache POI, Maven e GitHub Actions.

**Resultado técnico:** exportação automatizada para XLSX, coberta por testes e CI com Maven.

[Ver repositório](https://github.com/jctech9/Projeto-Exemplo-json-to-xlsx)

### Automations

Automações agendadas que consultam APIs HTTP e um feed RSS/Atom para gerar alertas de chuva e acompanhar novas publicações, enviando notificações pela Telegram Bot API.

**Tecnologias:** Python, GitHub Actions, APIs HTTP, Telegram Bot API e pytest.

**Resultado técnico:** workflows com retries, timeouts, cache de estado, validação TLS e testes automatizados.

[Ver repositório](https://github.com/jctech9/automations)

### UFCQ Monitor

Projeto pessoal de monitoramento de disponibilidade HTTP, latência, redirecionamentos e validade de certificado SSL, com histórico de verificações e incidentes dos últimos sete dias.

**Tecnologias:** Python, HTML, CSS, JavaScript, GitHub Actions, GitHub Pages e pytest.

**Resultado técnico:** geração e publicação periódica de uma página pública de status.

[Ver repositório](https://github.com/jctech9/ufcq-monitor) · [Acessar demonstração](https://jctech9.github.io/ufcq-monitor/)

### Watches

Aplicação web responsiva e instalável para gerenciar uma coleção de relógios, com autenticação por e-mail, persistência no Firestore e operações de cadastro, consulta, edição e exclusão.

**Tecnologias:** JavaScript, HTML, CSS, Firebase Authentication, Firestore, PWA e Service Worker.

**Resultado técnico:** CRUD autenticado com experiência responsiva e suporte de aplicação instalável.

[Ver repositório](https://github.com/jctech9/Watches) · [Acessar demonstração](https://jctech9.github.io/Watches/)

### Energy Consumption

Aplicação para registrar medições reais de energia e projetar consumo e custo mensal em diferentes modos de cálculo, com autenticação e persistência de dados.

**Resultado técnico:** cálculos de consumo e custo com validações, múltiplos modos de projeção e persistência remota.
